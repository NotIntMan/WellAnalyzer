# WellAnalyzer

Десктопное приложение для импорта CSV с данными по скважинам. Группирует интервалы по скважинам, валидирует данные и выводит сводку в таблице. Ошибки отображаются отдельным списком. Результат можно экспортировать в JSON.

## Запуск

Требования: .NET SDK 8

Для запуска проекта используйте команду:

```bash
dotnet run --project WellAnalyzer.App/WellAnalyzer.App.csproj
```

Для примера входных данных можно использовать файл `samples/test.csv`.

## Что ещё сделано

- **Неблокирующая загрузка** — парсинг CSV асинхронный, а валидация и расчёты вынесены в пул потоков, чтобы UI не блокировался на больших файлах.
- **Экспорт в JSON** — кнопка «Экспорт JSON» сохраняет сводку через диалог выбора файла.
- **Unit-тесты** — покрыты парсер, валидатор и калькулятор (xUnit + FluentAssertions).
- **Обработка ошибок без падений** — при сбое импорта или экспорта показывается модальное окно с описанием проблемы, приложение продолжает работать.
